Project Overview:

The aim of this project is to implement SLAM (Simultaneous Localization and Mapping) for a two dimensional world. Based on the map of the environment created, the location of the robot is tracked and the locations of the landmarks such as buildings are identified in the real time.

Image Source: [ Google ]

To summarize, for simultaneous localization and mapping (SLAM), information is gathered from a robot's sensors and motions over time, and then use that gathered information about measurements and motion to re-construct a map of the world.

Robot Creation and Movement while sensing the landmarks:

First, I have created a robot and allowed it to move around a 2D grid world.

Then, I have defined a sense function for the robot that allows it to sense landmarks in a given world so that the robot keeps track of the different landmarks that it sees in a 2D grid world .

Note: The robot motion and sensors have some uncertainty associated with them.

To create the robot, I have used following parameters:

  • size of world = 10.0
  • range at which landmarks are sensed = 5.0
  • noise in robot motion = 0.2
  • noise in the measurements = 0.2
