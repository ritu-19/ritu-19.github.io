Project Overview:

The main of the project is to apply basic machine learning concepts on the data collected for housing prices in the Boston, to predict the selling price of a new home.

Dataset Used

The dataset is taken from UCI Boston Housing . The dataset is modified a little bit.It consists of 489 data points, with each datapoint having 3 features and 1 target variable as described below:

  • Features

    • 1. RM: average number of rooms per dwelling 2. LSTAT: percentage of population considered lower status 3. PTRATIO: pupil-teacher ratio by town

  • Target Variable

    • 1. MEDV: median value of owner-occupied homes

Analysis and Methodology

Data Exploration:

First, Data Exploration is done to obtain important features and descriptive statistics about the dataset.

Second, Splitting the data into testing and training subsets, and determine a suitable performance metric for this problem.

Building Model, Analyzing and Visualizing:

Third, Analyzing and Visualizing the performance graphs for a learning algorithm with varying parameters and training set sizes.

Optimal Model:

Fourth, Picking the optimal model.

Testing / Prediction Accuracy:

Fifth, Finally, Testing the optimal model on a new sample and compare the predicted selling price to your statistics.
