Project Overview:

The main of the project is to apply basic machine learning concepts on the data collected for housing prices in the Boston, to predict the selling price of a new home.

Image Source: [ Google ]

Dataset Used

The dataset is taken from UCI Boston Housing . The dataset is modified a little bit.It consists of 489 data points, with each datapoint having 3 features and 1 target variable as described below:

  • Features

    • 1. RM: average number of rooms per dwelling 2. LSTAT: percentage of population considered lower status 3. PTRATIO: pupil-teacher ratio by town

  • Target Variable

    • 1. MEDV: median value of owner-occupied homes

Data Exploration:

  • First, dataset is explored and visualized to obtain important features and descriptive statistics about the dataset.

  • Second, I have Splitted the dataset into testing and training subsets, and have determined a suitable performance metric for solving this problem.

  • During data exploration, I observed the Statistics of the training and testing dataset such as, Maximum, Minimum, standard deviation etc price of the data.

  • I have used r2_score for measuring the performance.
